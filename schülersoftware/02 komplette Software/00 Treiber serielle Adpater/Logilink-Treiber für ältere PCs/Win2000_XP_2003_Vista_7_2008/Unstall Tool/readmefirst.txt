PL-2303 Driver Remover Program, Prolific Edition
Revision Note History
==================================================

System Requirement:
  . Supports the following Windows OSes:
	- Windows XP SP2 and above (32 & 64 bit)
	- Windows Vista (32 & 64 bit)
	- Windows 7 (32 & 64 bit)
  . USB host controller
  . Device using PL-2303H/HX/X version chip
 

Supported device ID and product strings:
  . VID_067B&PID_2303 for "Prolific USB-to-Serial Comm Port"


How to Use:
1. Unplug all PL-2303 devices. 
2. Use this program only if you are having problems uninstalling old and new drivers.
3. If you previously run a SETUP installer program, go first to Control Panel-Add/Remove Programs to remove the driver installer program. 
4. Run this program afterwards and reboot your computer. 
5. Install the new driver if OK.
